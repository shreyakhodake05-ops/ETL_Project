# ETL Data Pipeline Project

This project demonstrates an ETL (Extract, Transform, Load) pipeline using Python, Pandas, and Scikit-learn.

## Features
- Data extraction from CSV
- Data preprocessing (missing values, encoding, scaling)
- Data transformation
- Saving processed data

## Files
- etl_pipeline.py → main pipeline script
- data.csv → input dataset
- processed_features.csv → processed features
- processed_target.csv → target output
- processed_pipeline.pkl → saved pipeline

## How to run
python etl_pipeline.py
